# tdd-simplified
<h2>TDD (Test Driven Development)</h2>
<h3>The cycle of development:</h3>
<h4>Step 1</h4>
Red</br>
Write a test to fail.
<h4>Step 2</h4>
Green</br>
write a simple test to pass.
<h4>Step 3</h4>
Refactore</br>
Refactore your code.
